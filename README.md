# 仓库管理系统 java664

本项目是一款基于B/S架构的仓库管理系统，采用前后端分离的技术方案，旨在优化企业管理链路，涵盖客户、订单、生产、待检、入库、盘点及出库等环节。

## 技术栈

### 后端
- Spring Boot
- Spring Security
- MyBatis Plus
- Redis
- WebSocket

### 前端
- Vue
- Element UI
- ECharts
- Vuex
- WebSocket

## 功能模块

系统包括以下主要功能模块：
- 权限管理
- 用户管理
- 角色列表
- 订单管理
- 仓库管理
- 出库管理
- 入库管理
- 客户管理
- 统计管理
- 聊天室

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 后端：Java 1.8+，兼容主流数据库
- 前端：现代浏览器
